# 📊 Excel Sales Dashboard Project

## 📌 Overview

This project showcases an **interactive Excel dashboard** built to analyse and visualise sales data. The goal was to transform raw data into meaningful insights that support **data-driven decision-making**.

The dashboard enables users to explore trends, identify top-performing segments, and monitor key business metrics dynamically.

---

## 🎯 Objectives

* Clean and prepare raw sales data for analysis
* Build a structured data model in Excel
* Analyse key performance metrics using Pivot Tables
* Create an interactive dashboard for business insights

---

## 🧩 Dataset

* Source: Sample retail/sales dataset
* Records: Structured transactional data (e.g., sales, categories, dates)
* Features include:

  * Product / Category
  * Sales / Revenue
  * Dates (daily/monthly)
  * Customer / Region (if applicable)

---

## ⚙️ Tools & Techniques Used

### 🧹 Data Cleaning

* Removed duplicates and handled missing values
* Standardised date and numeric formats
* Used **Power Query** for data transformation

### 🔗 Data Modelling

* Structured data into tables
* Created calculated columns for KPIs
* Used **VLOOKUP / XLOOKUP** for data mapping

### 📊 Data Analysis

* Built **Pivot Tables** for:

  * Aggregations (SUM, COUNT, AVERAGE)
  * Trend analysis (monthly/yearly)
  * Segment comparisons

### 📈 Data Visualisation

* Designed an interactive dashboard with:

  * Bar charts (category comparisons)
  * Line charts (trend analysis)
  * KPI summaries
* Added **slicers** for filtering:

  * Time period
  * Category / Segment

---

## 🔍 Key Insights

* Identified top-performing categories/products
* Analysed sales trends over time
* Detected peak sales periods
* Compared performance across different segments

---

## 💡 Business Impact

This dashboard can help stakeholders:

* Make informed decisions on inventory and sales strategy
* Identify high-performing areas to maximise revenue
* Monitor performance trends quickly and efficiently

---

## 🚀 How to Use

1. Download the Excel file
2. Open in Microsoft Excel
3. Use slicers to filter data dynamically
4. Explore charts and KPIs for insights
